ABOUT THE GAME

Astral legends is a space shooter game developed by 8 students from BSCOE-12A1.

It was developed with four programming languages. HTML for structure, CSS for design, JavaScript for functionalities, and PHP for the database

It is heavily inspired by Space Invaders, an iconic video game that was released back in 1978 for arcades.

Astral Legends aims to rekindle that retro thrill with a modern twist!
